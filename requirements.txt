# Hemanth Kumar - AI/ML Developer Portfolio

A modern, responsive portfolio website showcasing AI/ML projects, skills, and experience.

## 🚀 Features

- **Modern Design**: Clean, dark theme with smooth animations
- **Responsive**: Mobile-first design that works on all devices
- **Interactive**: Custom cursor, hover effects, and smooth scrolling
- **Resume Download**: Direct PDF download functionality
- **Contact Integration**: Email integration with Gmail compose
- **Project Showcase**: Detailed project presentations with tech stacks
- **Timeline**: Interactive experience and achievement timeline

## 🛠️ Tech Stack

- **Framework**: Next.js 15.2.4 (React 19)
- **Styling**: Tailwind CSS 3.4.1
- **Animations**: Framer Motion 11.15.0
- **Icons**: React Icons 5.4.0, Lucide React 0.469.0
- **Language**: TypeScript 5
- **Build Tool**: Next.js with Turbopack

## 🚀 Getting Started

### Prerequisites
- Node.js 18.0.0 or higher
- npm 8.0.0 or pnpm 7.0.0 or higher

### Installation

1. Clone the repository
```bash
git clone https://github.com/HemanthKumar-CS/hemanth-portfolio.git
cd hemanth-portfolio
```

2. Install dependencies
```bash
npm install
# or
pnpm install
```

3. Run the development server
```bash
npm run dev
# or
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
# or
pnpm build
pnpm start
```
